# kalidocker

Based on the latest Kali docker. Used with the GNS3 net emulator (https://gns3.com).
A minimum sets of tools are installed to be used in the different GNS3 Labs.


## Tools installed
* bettercap
* cisco-global-exploiter 
* cisco-auditing-tool
* dsniff
* dnschef 
* dnsutils 
* ettercap-text-only
* fping
* hping3 
* hydra 
* metasploit-framework
* nmap
* snmp 
* sqlmap 
* telnet
* tshark 
* openssh-client
* python-scapy
* yersinia
